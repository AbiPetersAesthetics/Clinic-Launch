Fix Investment Tab Dividend / Distributable Profit Logic

The Monthly P&L Breakdown is now the source of truth for profit and cashflow.

Please update the Investment / 3-Year Performance Outlook tab so that the “Distributable Profit” line is calculated directly from the Monthly P&L Breakdown, rather than using a separate formula based on Winchester operating profit minus VAT.

Current issue

The Investment tab is currently calculating distributable profit using this logic:

combinedNet = (Winchester operating profit − Winchester VAT) + Bedhampton net profit
netPre = combinedNet − monthly loan repayment
distrib = netPre > 0 ? netPre : 0

This is incorrect for the Investment tab because the Monthly P&L already calculates VAT, variable costs, fixed costs, salary, Bedhampton net, loan repayments, and project costs month by month.

The current Investment tab appears to understate FY27/28 distributable profit by deducting VAT again or by using a different profit definition from the Monthly P&L.

Required change

Please update the Investment tab so that the annual dividend/distributable figure is based on the Monthly P&L column:

Net after Salary

This column should be treated as the best current proxy for:

Indicative maximum dividend capacity before corporation tax and board/accountant adjustments
New calculation

For each financial year, calculate:

indicativeDividendCapacity = sum(month.netAfterSalary for all months in that financial year where month.netAfterSalary > 0)

Use the August–July financial year:

FY26/27 = Aug 2026 to Jul 2027
FY27/28 = Aug 2027 to Jul 2028
FY28/29 = Aug 2028 to Jul 2029
Important labelling change

Do not label this line simply as “Distributable Profit”, because that implies a legally confirmed dividend figure.

Please rename the line to:

Indicative Max Dividend Capacity

or:

Indicative Pre-Tax Dividend Capacity

Then add a small note underneath the table:

Indicative Max Dividend Capacity is calculated from the Monthly P&L “Net after Salary” line and represents the maximum theoretical amount available before corporation tax, retained earnings checks, working capital decisions, and board/accountant approval. Actual lawful dividends must be confirmed later from statutory accounts and available distributable reserves.

Logic rules

Please ensure:

The Investment tab pulls directly from the same monthly objects/data array used by the Monthly P&L Breakdown.
No separate hidden VAT deduction is applied inside the Investment tab.
VAT should only be deducted once, inside the Monthly P&L calculation.
Loan repayments should not be deducted twice.
The “Capital” column should continue to work as the running cash/capital balance.
The “Indicative Max Dividend Capacity” should be calculated from the Monthly P&L “Net after Salary” values, not from gross revenue or operating profit.
The label should make clear this is not a legally confirmed distributable reserve figure.
Add a tooltip or footnote explaining that true distributable reserves will later need corporation tax/accountant/statutory accounts treatment.
