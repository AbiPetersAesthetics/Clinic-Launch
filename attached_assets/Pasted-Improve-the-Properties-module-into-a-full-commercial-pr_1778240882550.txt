Improve the Properties module into a full commercial property acquisition and clinic intelligence engine for aesthetics clinic expansion planning.

The Properties section should no longer feel like a static card list. It should feel like a strategic command centre used to source, evaluate, compare, score, negotiate and select commercial clinic premises.

CORE OBJECTIVE:
Allow the user to search for, import, analyse, compare and select commercial properties, then automatically feed the chosen property into the wider Clinic Launch OS financial and project systems.
